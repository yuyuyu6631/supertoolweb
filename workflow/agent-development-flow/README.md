# Agent Development Flow

这是一个面向 agent 的通用开发工作流，用来把需求改动沉淀成可重复执行、可验证、可汇报的标准流程。

适用场景：
- 新功能开发
- Bug 修复
- 重构与优化
- 文档和脚手架更新

推荐使用顺序：
1. 先读 [WORKFLOW.md](./WORKFLOW.md)
2. 执行前对照 [CHECKLIST.md](./CHECKLIST.md)
3. 需要复用时直接填 [PROMPT_TEMPLATE.md](./PROMPT_TEMPLATE.md)
4. 完成后按 [OUTPUT_TEMPLATE.md](./OUTPUT_TEMPLATE.md) 汇报
