# Workflow Library

这个目录用于沉淀可复用的 AI 工作流规范。

目标：

- 把一次性任务沉淀成可重复执行的标准流程
- 为后续 AI 协作提供统一输入、步骤、输出和验收口径
- 减少“每次都重新解释”的沟通成本

当前已提供的工作流：

- [doc-current-state-alignment/README.md](./doc-current-state-alignment/README.md)
- [demo-video-production/README.md](./demo-video-production/README.md)
- [agent-development-flow/README.md](./agent-development-flow/README.md)

使用原则：

- 工作流描述的是“怎么做”
- 业务文档描述的是“当前是什么”
- 如果工作流与代码事实冲突，以代码和当前任务目标为准，并及时回写工作流
