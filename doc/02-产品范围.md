# 产品范围

## 当前已实现的能力

当前仓库已经落地以下公开页面与模块：

1. 首页 `/`
2. 工具目录页 `/tools`
3. 工具详情页 `/tools/[slug]`
4. 榜单页 `/rankings`
5. 场景列表页 `/scenarios`
6. 场景详情页 `/scenarios/[slug]`
7. 登录注册页 `/auth`
8. 首页和目录页的真实 API 数据读取
9. 基础账号认证能力：注册、登录、登出、读取当前登录用户
10. 目录筛选能力：搜索、分类、标签、状态、视图、排序、分页
11. 基础抓取任务入口：`POST /api/crawl/jobs`

## 当前明确不在本轮范围内的能力

以下能力当前未落地，或仅停留在未来规划：

- 评论发布与社区互动
- 收藏历史与个人中心
- 支付、订单、会员体系
- 多轮对话历史和会话归档
- 后台管理界面
- 真实抓取编排与审核后台闭环
- 真实福利/商业化转化链路
- SEO 专题内容生产系统
- 第三方 OAuth 社交登录

说明：

- 登录注册已经进入当前范围，但仍属于基础版账号能力
- 页面内容当前以真实后端接口为主，不再以浏览器侧 mock API 为现状基础
- 分类 API 已存在，但当前前端没有独立 `/categories/[slug]` 页面

## 当前站点信息架构

```text
首页 /
├── 搜索入口
├── 常用分类
├── 精选工具
└── 快速跳转到工具目录

工具目录 /tools
├── 搜索
├── 视图切换
├── 分类筛选
├── 状态筛选
├── 标签筛选
└── 分页列表

工具详情 /tools/[slug]
└── 单个工具的完整信息展示

榜单 /rankings
└── 多个榜单分组展示

场景列表 /scenarios
└── 场景入口列表

场景详情 /scenarios/[slug]
└── 单个场景说明与工具组合

认证 /auth
├── 登录
├── 注册
└── 已登录状态展示与退出
```

## 当前边界说明

- 当前项目已经不是“只有静态演示”的纯前端 Demo
- 当前项目也还不是完整商业化产品，仍处于 MVP/MVO 过渡阶段
- 文档中的未来能力仅作规划，不应误认为已经在代码中落地
