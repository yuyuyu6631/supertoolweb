# AI 工具评测平台文档索引

本目录用于沉淀当前项目的产品说明、实现基线、接口约束、数据结构和阶段性记录。

## 当前基线

- 当前前端真实实现为 `Next.js 15 + React 19 + Tailwind CSS 4`
- 当前后端真实实现为 `FastAPI + SQLAlchemy 2 + Alembic + MySQL`
- 前端已接入真实 API，请求默认指向 `http://localhost:8000`
- 当前公开页面包含首页、工具目录、工具详情、榜单、场景页和登录注册页
- 当前已具备基础账号认证能力，使用服务端会话和 HttpOnly Cookie
- 文档中凡标记为“未来规划”的内容，仅代表目标方向，不代表当前代码状态
