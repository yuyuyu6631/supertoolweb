# 05 接口规范

## 1. 当前实现说明

当前前端通过 `NEXT_PUBLIC_API_BASE_URL` 直接请求 FastAPI 服务，默认地址为 `http://localhost:8000`。  
目录、榜单、场景、推荐、抓取和认证均已使用真实后端路由，不再以浏览器侧 `mock API` 作为当前基线。

## 2. 当前公开 API

### 2.1 目录与详情

- `GET /api/tools`
- `GET /api/tools/{slug}`
- `GET /api/categories`
- `GET /api/categories/{slug}/tools`
- `GET /api/rankings`
- `GET /api/rankings/{slug}`
- `GET /api/scenarios`
- `GET /api/scenarios/{slug}`

`GET /api/tools` 支持以下查询参数：

- `q`
- `category`
- `tag`
- `status`
- `sort`
- `view`
- `page`
- `page_size`

### 2.2 推荐

- `POST /api/recommend`

请求体：

```json
{
  "query": "我想找做 PPT 的 AI",
  "scenario": "ppt",
  "tags": ["演示", "生成"],
  "candidateSlugs": []
}
```

字段说明：

- `query`：必填，最少 2 个字符
- `scenario`：可选，场景标识
- `tags`：可选，标签列表
- `candidateSlugs`：可选，候选工具 slug 列表

成功响应：

```json
[
  {
    "tool_id": 3,
    "name": "Gamma",
    "slug": "gamma",
    "url": "https://gamma.app",
    "summary": "适合快速生成演示文稿和视觉化内容。",
    "tags": ["PPT", "演示", "生成"],
    "reason": "更适合快速完成演示稿初稿。",
    "score": 9.0,
    "logoPath": "/logos/gamma.png"
  }
]
```

### 2.3 抓取任务

- `POST /api/crawl/jobs`

当前实现说明：

- 当前接口仍是轻量入口
- 由 `crawler_service.build_mock_snapshot` 返回抓取快照结果
- 已存在定时任务调度与数据表，但还未形成完整审核发布闭环

### 2.4 认证

- `POST /api/auth/register`
- `POST /api/auth/login`
- `POST /api/auth/logout`
- `GET /api/auth/me`

注册请求体：

```json
{
  "username": "demo_user",
  "email": "demo@example.com",
  "password": "password123",
  "confirmPassword": "password123",
  "agreed": true
}
```

登录请求体：

```json
{
  "identifier": "demo@example.com",
  "password": "password123"
}
```

用户响应体：

```json
{
  "id": 1,
  "username": "demo_user",
  "email": "demo@example.com",
  "status": "active",
  "createdAt": "2026-03-31T10:00:00Z"
}
```

认证实现说明：

- 使用 HttpOnly Cookie 维护服务端会话
- `/api/auth/me` 用于恢复登录态
- `/api/auth/logout` 成功时返回 `204 No Content`

## 3. 异常响应

当前后端主要使用 FastAPI 标准错误结构：

```json
{
  "detail": "错误说明"
}
```

或字段级错误：

```json
{
  "detail": {
    "email": "这个邮箱已经注册过账号了。"
  }
}
```

典型情况：

- 参数错误：`422`
- 未登录或会话失效：`401`
- 资源不存在：`404`
- 冲突：`409`
- 服务端异常：`500`

## 4. 前端当前依赖的关键返回结构

- 工具目录：`ToolsDirectoryResponse`
- 工具详情：`ToolDetail`
- 榜单列表：`RankingSection[]`
- 场景列表/详情：`ScenarioSummary[]` / `ScenarioSummary`
- 推荐结果：`RecommendItem[]`
- 当前用户：`AuthUser`

## 5. 维护约束

- 文档中的接口以 `apps/api/app/api/routes/*.py` 为准
- 文档中的字段结构以 `apps/api/app/schemas/*.py` 为准
- 若接口新增字段或改名，需要同步更新本文件和 `doc/06-数据结构.md`
